# How to Link Google Ads Account to Manager Account

## Step-by-Step Instructions

### Option 1: Link Account from Manager Account (Recommended)

1. **Log into your Manager Account (MCC)**
   - Go to: https://ads.google.com/
   - Make sure you're in the Manager Account view (not a regular account)
   - You'll see "All accounts" or a list of accounts in the top bar

2. **Navigate to Account Access**
   - Click on the account selector (top left, shows current account name)
   - Click "Account access and security" or "Access and security"
   - OR go directly to: https://ads.google.com/aw/accountaccess

3. **Add Account**
   - Look for a button that says "Link existing account" or "+ Link account"
   - Click it

4. **Enter Your Customer ID**
   - You'll be prompted to enter a Customer ID
   - Enter: **6780818726053668** (your Google Ads account ID)
   - Or enter the account email if prompted

5. **Send Request**
   - Click "Send request" or "Link account"
   - A link request will be sent to the account

6. **Accept the Request** (from the regular account)
   - Log into your regular Google Ads account (6780818726053668)
   - You should see a notification or email about the link request
   - Go to: Tools & Settings → Access and security
   - Find the pending request from the Manager Account
   - Click "Accept" or "Approve"

---

### Option 2: Request Access from Regular Account

1. **Log into your Regular Google Ads Account**
   - Go to: https://ads.google.com/
   - Make sure you're in account: **6780818726053668**

2. **Navigate to Access and Security**
   - Click Tools & Settings (wrench icon)
   - Under "Setup", click "Access and security"
   - OR go to: https://ads.google.com/aw/accountaccess

3. **Request Manager Access**
   - Look for "Managers" section
   - Click "Request access" or similar button
   - Enter the Manager Account ID or email
   - Send the request

4. **Accept from Manager Account**
   - Switch to your Manager Account
   - You'll see a notification about the request
   - Go to Account Access
   - Approve the request

---

### Option 3: Direct Link (If You Have Admin Access)

If you're the owner/admin of both accounts:

1. **From Manager Account:**
   - Go to: Account Access and Security
   - Click "Link existing account"
   - Enter Customer ID: **6780818726053668**
   - The link should be instant if you have the right permissions

---

## Verify the Link

After linking, verify it worked:

1. **In Manager Account:**
   - Click the account selector (top left)
   - You should see account **6780818726053668** in the list
   - You can switch to it directly

2. **Access API Center:**
   - While in the Manager Account
   - Go to: Tools & Settings → API Center
   - You should now have access!
   - The API Center will work for all linked accounts

---

## Important Notes

- **Both accounts must use the same email** OR have appropriate permissions
- The account owner/admin needs to approve the link request
- Once linked, the Manager Account can access all accounts under it
- You can link multiple accounts to one Manager Account
- Linking is **free** - no additional cost

## Troubleshooting

### "You don't have permission to link this account"
- Make sure you're an admin on the Manager Account
- Make sure you're an admin on the regular account
- Try using the same Google account for both

### "Account already linked"
- The account might already be linked to another Manager Account
- Check if you can access it from the Manager Account dropdown

### "Request not appearing"
- Check email notifications
- Look in the Manager Account's "Account access" section
- Make sure you're checking the right account

### "Can't find Account Access"
- Try this direct link: https://ads.google.com/aw/accountaccess
- Make sure you're in a Manager Account (not regular account)
- The option might be under "Tools & Settings" → "Setup"
