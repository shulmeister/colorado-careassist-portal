## Documentation Maintenance

**When to archive**:
- Status reports older than 3 months
- Deployment summaries for completed releases
- Documentation superseded by newer guides

**When to update setup guides**:
- API changes from third-party services
- New integration features
- Security updates
- Breaking changes

**When to update main README**:
- New features added
- Architecture changes
- Environment variable changes
- Deployment process changes
