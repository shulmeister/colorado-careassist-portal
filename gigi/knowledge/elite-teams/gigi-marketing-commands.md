# Gigi Marketing Management Commands

## Daily Operations

### Morning Standup
```
"Gigi, run marketing team daily standup:
1. Message each marketing team member for status update
2. Collect yesterday's deliverables and today's priorities
3. Identify any blockers or support needed
4. Compile team status report
5. Flag any missed deadlines or concerns"
```

### Performance Check
```
"Gigi, pull marketing performance dashboard:
1. Get Google Ads ROAS from last 7 days
2. Check organic traffic trends (GA4)
3. Review email campaign metrics (Brevo)
4. Monitor social engagement (Facebook/Instagram APIs)
5. Compare vs. targets and alert on issues"
```

## Skill Module Enforcement

### Campaign Validation
```
"Gigi, validate upcoming marketing campaign:
1. Confirm relevant skill module was consulted
2. Check framework/checklist completion
3. Verify tracking implementation
4. Ensure deliverable meets quality standards
5. Approve for launch or flag issues"
```

### SEO Audit Trigger
```
"Gigi, initiate monthly SEO audit for CCA:
1. Use seo-audit skill module framework

3. Generate technical SEO checklist
4. Assign prioritized action items to Search Specialist
5. Set deadlines and track completion"
```

## Team Accountability

### Weekly Review
```
"Gigi, conduct marketing team weekly review:
1. Compare deliverables completed vs. assigned
2. Review metrics vs. targets by team member
3. Identify top performers and improvement areas
4. Generate performance summary for Jason
5. Set next week's priorities and assignments"
```

### Escalation Protocol
```
"Gigi, check for marketing team escalations:
1. Flag agents with 2+ missed deliverables
2. Alert on metrics declining 3+ weeks
3. Identify resource/support needs
4. Recommend intervention actions
5. Notify Jason of critical issues requiring attention"
```
