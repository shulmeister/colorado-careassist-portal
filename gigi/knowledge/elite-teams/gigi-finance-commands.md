# Gigi Finance Team Management Commands

## Daily Operations

### Financial Pulse Check
```
"Gigi, run finance team daily pulse check:
1. Check cash position using qb balance
2. Review AR aging with qb customers
3. Monitor new invoices and payments received
4. Check AP payment schedule and timing
5. Flag any urgent financial issues or cash alerts"
```

### Cash Flow Monitoring
```
"Gigi, monitor daily cash flow:
1. Run qb balance for current cash position
2. Calculate days of operating expenses available
3. Review incoming payments and timing
4. Check outgoing payment schedule
5. Alert if cash runway drops below 6 months"
```

## QuickBooks CLI Automation

### Morning Financial Dashboard
```
"Gigi, generate morning financial dashboard:
1. Pull cash position: qb balance
2. Review customer balances: qb customers
3. Check recent invoicing: qb invoices
4. Monitor vendor balances: qb vendors
5. Compile daily financial status report"
```

### AR Aging Analysis
```
"Gigi, analyze AR aging for CCA:
1. Run qb customers for detailed aging
2. Identify accounts > 30 days outstanding
3. Flag accounts > 60 days for immediate action
4. Calculate DSO and trend analysis
5. Assign collection priorities to AR specialist"
```
